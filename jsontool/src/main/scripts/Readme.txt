本工具可实现：
1. 从标准输入读入json串、以缩进、美化形式输出。使用时指定-p参数。
2. 从标准输入读入json串、以紧凑形式输出。使用时指定-P参数。
3. 使用命令行重定向方式从文件读入json串、以紧凑或者美化形式输出。
          使用需指定文件字符编码参数、即:-c charsetname。
   若不指定文件字符编码、则默认使用utf8编码。

运行方法：
在命令行下运行start.cmd

使用例子：
1. 从标准输入读入json串、以缩进、美化形式输出
   start.cmd -p 

2. 从标准输入读入json串、以紧凑形式输出。使用时指定-P参数。
   start.cmd -P

3. 使用命令行重定向方式从文件读入json串、以紧凑形式输出。
   start.cmd -P -c utf8 <myjson.json
